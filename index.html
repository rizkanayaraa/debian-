Cloud Computing

Rizka Nayara | 211011401190 | 04TPLP010

Cloud Computing adalah teknologi yang memungkinkan pengguna untuk mengakses server, penyimpanan, dan aplikasi melalui internet tanpa perlu memiliki perangkat fisik sendiri.

Pertemuan 1

۶ৎ Instalasi Debian ۶ৎ

Pada pertemuan ini dilakukan instalasi sistem operasi Debian sebagai dasar dalam membangun server. Debian dipilih karena memiliki tingkat stabilitas yang tinggi dan banyak digunakan dalam lingkungan server maupun cloud computing. Proses instalasi dilakukan melalui virtual machine agar dapat mensimulasikan penggunaan server secara nyata tanpa membutuhkan perangkat fisik tambahan.

  • Menyiapkan file ISO Debian
  • Membuat Virtual Machine
  • Booting dan instalasi sistem
  • Konfigurasi hostname
  • Membuat user
  • Instalasi sistem dasar
Pertemuan 2

۶ৎ Konfigurasi IP Address ۶ৎ

Pada tahap ini dilakukan konfigurasi IP Address untuk memberikan alamat tetap (static IP) pada server. Hal ini penting agar server dapat diakses secara konsisten dalam jaringan. Dengan menggunakan IP statis, server tidak akan berubah alamatnya sehingga memudahkan proses pengelolaan dan koneksi antar perangkat.

  • Cek interface (ip a)
  • Edit file network
  • Set IP Address
  • Restart networking
  • Test koneksi
Pertemuan 3

۶ৎ DNS Server ۶ৎ

Pada pertemuan ini dilakukan konfigurasi DNS Server yang berfungsi untuk menerjemahkan nama domain menjadi alamat IP. Dengan adanya DNS, pengguna tidak perlu mengingat alamat IP server, cukup menggunakan nama domain yang lebih mudah diingat. DNS server sangat penting dalam infrastruktur jaringan dan cloud computing.

  • Install bind9
  • Konfigurasi file named.conf
  • Membuat zone domain
  • Setting forward & reverse DNS
  • Restart service DNS
Pertemuan 4

۶ৎ Konfigurasi Web Server ۶ৎ

Pada tahap ini dilakukan konfigurasi web server menggunakan Apache. Web server berfungsi untuk menampilkan website agar dapat diakses melalui browser. Setelah Apache terinstal, dilakukan pengujian dengan mengakses alamat IP server dan melakukan deploy file HTML sebagai halaman website.

  • Install apache2
  • Cek status server
  • Akses melalui browser
  • Edit file HTML
  • Deploy website
